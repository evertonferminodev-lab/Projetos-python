# Sistema de cadastro (CRUD)

Sobre o projeto

Este projeto consiste em um sistema simples de cadastro desenvolvido em Python utilizando o conceito de CRUD. O objetivo é praticar lógica de programação, organização de código com funções e criação de menus interativos no terminal.

O sistema permite:

Cadastrar usuários

Listar usuários cadastrados

Atualizar informações

Remover usuários

Este projeto foi desenvolvido como parte do meu aprendizado em programação, com foco na construção de uma base sólida para desenvolvimento backend.
